Latitude

Summary Lattitude VS X

The purpose of this project was to analyze the weather changes as you get closer to the equator.To accomplish this analysis, we first pulled data from OpenWeatherAPI to assemble a data set of 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. Factore we looked included : temperature, Cloudiness, wind speed and humidity. The sitre provides the source data and visualizations created as part of this analysis, as well as explanations and descriptions of any trends and corelations witnessed
